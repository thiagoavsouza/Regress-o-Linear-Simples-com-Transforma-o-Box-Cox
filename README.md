# Regressão Linear Simples com Transformação Box-Cox — Idade x Comprimento do Bebê

## Descrição do Projeto

Este projeto aplica **Regressão Linear Simples (OLS)** com **Transformação Box-Cox**
para modelar a relação entre a variável **Idade (semanas)** e o **Comprimento do bebê**.

O objetivo é:
- Ajustar um modelo linear após correção de normalidade,
- Validar as premissas da regressão,
- Obter uma predição confiável do comprimento médio para uma idade específica,
- Realizar o cálculo inverso da transformação para retornar à escala original.

## Etapas do Projeto

1. Importação e análise inicial dos dados  
2. Diagnóstico do modelo linear inicial  
3. Transformação Box-Cox  
4. Avaliação do modelo OLS transformado  
5. Predição do comprimento
